# SAIDify example

This repository shows an example of how to use the [SAIDify](https://github.com/kentbull/saidify) library.

See [index.ts](./index.ts) for the example code.


## to run
### typescript thing
`npm install -D typescript`

### generate js files
`npx tsc`

### run js file
`node index.js`
